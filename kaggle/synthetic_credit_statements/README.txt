Credit Consumer and SMEs Financial Behavior Analysis
Overview
This project focuses on analyzing and understanding the financial behavior of credit consumers and small to medium-sized enterprises (SMEs). The primary goal is to provide insights that can support targeted marketing efforts for credit bureau members.

Objectives
Customer Segmentation: Utilize clustering algorithms to segment credit consumers based on their financial attributes.
Marketing Strategies: Develop personalized marketing strategies for different customer segments.
Outcome Improvement: Enhance the effectiveness of marketing campaigns and increase engagement.
Project Structure
Data Exploration: Explore and analyze the datasets for credit consumers and SMEs.
Correlation Analysis: Investigate relationships between various financial attributes.
Visualization: Create visualizations to gain insights into customer behavior.
Clustering: Apply clustering algorithms to segment credit consumers.
Marketing Strategies: Devise personalized marketing strategies for each cluster.
Outcome Evaluation: Assess the impact of implemented strategies on customer engagement.
Data Sources
Credit Consumers Dataset
SMEs Dataset
Technologies Used
Python
Pandas, Matplotlib, Seaborn for data analysis and visualization
Plotly, Scikit-learn for interactive plots and clustering algorithms
How to Use
Ensure you have the necessary Python libraries installed. You can install them using:


pip install pandas matplotlib seaborn plotly scikit-learn
Open the Jupyter Notebook and execute each cell sequentially to run the analysis.

Explore the visualizations and insights generated throughout the notebook.

Results
The project aims to provide actionable insights for credit bureau members to improve their marketing strategies and engagement with credit consumers and SMEs.
